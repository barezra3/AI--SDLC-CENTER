# AI--SDLC-CENTER
# 🛡️ AI-SDLC Command Center: Enterprise Project Management Agent

An autonomous Project Management framework that synchronizes with **Jira Service Management** to analyze, risk-assess, and automate SDLC processes using Multi-Agent AI architecture.

## 🚀 Overview
Inspired by operational efficiency and the need for automation in complex systems (8200 mindset), this tool transforms raw Jira data into actionable insights. It doesn't just display tasks; it understands them using a "Board of Directors" of AI Agents.

## ✨ Key Features
- **Jira Integration:** Real-time data synchronization using Python APIs.
- **AI Board of Directors:** - **CISO Agent:** Scans task descriptions for security vulnerabilities and compliance.
  - **Delivery Manager:** Identifies timeline bottlenecks and predicts project risks.
  - **Product Manager:** Automatically generates PRDs and refines business requirements.
- **Automated Outputs:** Instantly generates Release Notes, Risk Reports, and Interactive Gantt Visualizations.
- **Dockerized Architecture:** Containerized for seamless deployment and consistent environments.

## 🛠️ Tech Stack
- **Language:** Python 3.10+
- **AI Orchestration:** CrewAI / LangChain
- **LLM Support:** Groq (LPU Inference) / GPT-4o
- **Containerization:** Docker & Docker-Compose
- **Dashboard/UI:** Streamlit
